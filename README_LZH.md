## 概述

Neo Eco AE Extension，乃 [Eco AE Extension](https://github.com/sddsd2332/NovaEngineering-ECOAEExtension) 之高版本移植也，增自[新星工程：世界](https://www.mcmod.cn/modpack/784.html)之 MMCE 多方塊結構。既獲原作者之授權，本項目以 GNU 通用公共許可證第三版（GPLv3）發佈之。

## Neo Eco AE Extension提供了什么?

Neo Eco AE Extension 為 Applied Energistics 2 引入三種大型核心組件，分合成、存儲與計算為三獨立之多方塊結構。每一結構各具三等，第等遞升，其性能較前級皆有顯著之增益。

此外，本模組復設諸多高性能組件，大幅增益自動化系統之便用與操持之舒適，使汝自動化之行，如呼吸然易。~~核電，輕而易舉啊~~

## 重要提示

Neo Eco AE Extension 與 Eco AE Extension 之間，惟經授權之移植關係而已，於項目層面無任何直接隸屬或依存之關係。二項目各由不同之開發團隊獨立維護與開發。

毋將關於 Neo Eco AE Extension 之問題、錯誤回報或技術支援之請求提交於 Eco AE Extension 項目，否則相關之事將不得有效處置，且或致 Eco AE Extension 團隊之維護與開發受無謂之擾。
