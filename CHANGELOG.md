# 更新日志

## 1.21.1-1.3.2 (2026-03-15)

对比版本：`1.21.1-1.3.1`

### 改进
- 渲染链路：为 computation 模型查询增加安全回退与节流告警；为固定渲染公共层增加 missing-model 防线，降低黑紫贴图扩散风险。（工作区加固）

### 修复
- 计算系统 GUI：修复 AE2 合成状态与 ECO 计算/合成主机在任务开始、结束、取消、超频切换后的存储与线程显示不同步问题，改为事件驱动刷新并加入同 tick 合并更新。（工作区修复）
- 存储矩阵驱动器：修复放入存储矩阵后显示黑紫缺失贴图的问题；补充 builtin 模型硬回退并修正相关模型回退链路。（工作区修复）

## 1.21.1-1.2.0 (2026-02-25)

对比版本：`1.21.1-1.1.1`

### 新增
- 集成工作站：新增“输出面选择/允许输出面”面板，用于配置哪些面允许自动输出。（4c0a8cc）
- 配置项：新增 `postCraftingEvent`，用于控制合成子系统完成配方时是否发布原版合成事件 `ItemCraftedEvent`。（f71aaaa）
- EMI：新增集成工作站的 EMI 显示支持。（e9318e8）
- EMI：新增冷却配方的 EMI 显示支持。（06aa841）

### 改进
- 集成工作站：自动输出相关行为保留内存卡配置。（20c84de）
- 注册流程：`ECOCellType` 支持通过 `NERegistrate` 注册，便于扩展与集成。（624994e）
- 依赖：更新 `ldlib2`。（54cb0ef）
- 资源：更新多项物品贴图。（5468da7）

### 修复
- C 系列：修复“晶振驱动器线缆消失”的问题。（51b9d08）
- C 系列：修复崩溃问题。（b0cbe30）
- 渲染/区块：修复 #48，使用 `RenderChunkRegion` 获取 `blockEntity` 替代 `ClientLevel`。（0ae339e）
- 同步：修复若干同步问题，并尝试修复磁盘 stack 同步问题。（6f017bf、09d09a8）

- 化学品存储：带辐射的化学品不再允许存入 ECO 的硬盘（化学存储单元）。（2c915ee）
