# 更新日志

## 1.0.8+mc1.20.1 (2026-06-13)

对比版本：`1.0.7+mc1.20.1`

### 新增
- 结构终端加入单页结构操作流程，支持下一次 `Shift+右键` 主机时执行搭建、镜像或拆除，并保留镜像预览、成型预览和预览状态记忆。
- C 系列计算主机 UI 新增真实下单任务列表，直接展示活跃 ECO CPU 正在执行的任务、数量、状态和进度。
- F 系列合成主机 UI 新增/完善合成任务卡片、滚动条与 tooltip，按 FX 工作核心内的实际运行任务聚合显示。
- L 系列存储主机右侧矩阵区域加入矩阵卡片 tooltip 颜色分级，按容量占用率高亮已用类型和已用字节。
- 结构终端多方块预览支持更精细的 `Shift` 缩放，并新增镜像/成型预览控制。

### 改进
- 大幅重构 F/C/L 系列主机 LDLib UI 布局：F 系列边框与结构模块预览区域修正，C 系列任务区与物品栏重新对齐，L 系列矩阵预览区上移并与物品栏对齐。
- 优化结构终端 UI：移除无用的建造辅助页，将搭建提示和操作按钮整合到主预览页，并调整按钮文案与位置。
- 结构终端多方块预览会记忆旋转、缩放、镜像、成型预览与层级等状态，重新打开后不再回到初始状态。
- 多方块预览的镜像与成型渲染复用实际主世界成型逻辑，减少镜像预览错位。
- 计算与合成任务状态同步协议扩展，C/F 系列 UI 可同步真实任务条目和进度数据。
- 移除 GeckoLib 依赖，清理相关渲染桥接，减少发布包依赖面。
- EMI 兼容处理改进，避免特定屏幕叠加时的 UI 冲突。

### 修复
- 修复结构终端预览区内按 `G` 打开并关闭 GuideME 后可能崩溃的问题。
- 修复 IWS 自动弹出导致合成产物未回到网络、可能在弹出阶段消失的问题。
- 修复 ECO-CI 超导晶阵传输总线模型中间冷却条渲染异常。
- 修复 C 系列主机下单后 UI 不显示对应任务的问题。
- 修复 F 系列合成任务显示为大量 `x1` 条目的问题，现在会按 FX 工作核心聚合。
- 修复 L 系列矩阵面板与滚动条对齐问题，并提升矩阵 tooltip 可读性。
- 修复 1.20.1 计算系统与 ExtendedAE Plus 虚拟合成卡不兼容导致最后一次虚拟下发后任务卡住的问题。
- 修复若干控制器 UI 统计读序和同步细节，减少界面打开时的旧值、跳变或显示重叠。

## 1.0.4+mc1.20.1 (2026-06-06)

### 新增
- 智能样板总线新增 1-8 页容量配置，默认 2 页，每页 63 个样板槽。
- 智能样板总线 GUI 新增服务端同步分页，支持独立翻页、当前页 Shift 移动和完整槽位同步。
- 恢复 AAE 与 EAE 的配方联动。

### 改进
- FastPath 增加每 tick 256 次累计批量预算，保留单批默认上限 64。
- 扩容后的样板库存兼容旧 63 槽 NBT；配置缩小时保留并开放包含旧样板的高页。
- 修正 JEI 冷却系统与集成工作站配方框显示，并恢复多方块预览拖动和滚轮缩放。

### 修复
- FastPath 批量任务同时受 Worker 剩余槽位和控制器总剩余线程限制，并在最终接受时再次校验。
- 特殊配方、未知配方和无法读取底层配方的样板不再进入 FastPath。
- 批量拒绝时回滚额外输入并继续普通逐份执行，避免输入、能量和任务统计重复处理。
- 修复集成工作站插入加速卡时意外同步打开 GUI，Shift 操作可正常阻止界面打开。
